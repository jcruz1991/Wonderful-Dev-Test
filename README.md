# Dev Test

Wonderful UI/UX challenge.

## View Page

1. Open html file
    1. Clone repository
    2. Enter root directory of folder
    3. Open index.html
2. Run server (must have node installed on machine)
    1. Clone repository
    2. Enter root directory of folder
    3. run "npm install"
    4. run "npm run start"
3. View website on Heroku: [Heroku Link](https://tranquil-tor-93687.herokuapp.com/ "Herku Link")
